# Registration-Sidecar

### Das Registration-Sidecar reicht folgende Environments an den ETCD-Service weiter:
    - APP_ID
    - APP_NAME
    - APP_TITLE
    - APP_SHORT_DESCRIPTION
    - APP_DESCRIPTION
    - APP_CATEGORY
    - APP_DOMAIN_SERVICE
    - APP_DOMAIN_PORT
    - APP_DOMAIN_PATH
    - APP_ICON_SERVICE
    - APP_ICON_PORT
    - APP_ICON_PATH
    - APP_API_SERVICE
    - APP_API_PORT
    - APP_API_PATH
    - SSO_ACTIVATED
    - HOST_IP
    - SSO_APP_PREFIX

<WERT> muss von ihnen ersetzt werden.
Weiter Besipiele sind in der App Pacman zu sehen.
